# 仁济药品库

📰 API文档 

---

## 1. 药品库专用检索

### Request URL

```
http://172.20.137.65:13943/search_drugs
```

---

## 请求参数

| 字段名         | 类型      | 必填 | 默认值      |
| ----------- | ------- | -- | -------- |
| query       | string  | 是  | -        |
| top_k       | integer | 否  | 3        |
| search_type | string  | 否  | "vector" |

---

## 响应参数

| 字段名         | 类型      | 描述        |
| ----------- | ------- | --------- |
| query       | string  | 原始查询文本    |
| search_type | string  | 实际使用的检索类型 |
| top_k       | integer | 请求的返回数量   |
| count       | integer | 实际返回结果数量  |
| results     | array   | 检索结果列表    |

---

## results 中参数介绍

| 字段名                 | 类型     | 描述                  |
| ------------------- | ------ | ------------------- |
| similarity          | float  | 相似度分数（0~1，向量检索归一化后） |
| drug_name           | string | 药品名称                |
| indications         | string | 适应症                 |
| related_diseases    | string | 相关疾病                |
| adverse_reactions   | string | 不良反应                |
| contraindications   | string | 禁忌                  |
| precautions         | string | 注意事项                |
| pregnancy_lactation | string | 孕妇及哺乳期用药            |
| pediatric_use       | string | 儿童用药                |
| geriatric_use       | string | 老年用药                |
| interactions        | string | 药物相互作用              |
| pharmacology        | string | 药理毒理                |
| text                | string | 原始文本片段（包含上述所有字段描述）  |

## 请求代码示例

```python
import requests

url = "http://172.20.137.65:13943/search_drugs"

data = {
    "query": "消化酶",
    "top_k": 3,
    "search_type": "vector"
}

response = requests.post(url, json=data)
results = response.json()

print(results)
```
